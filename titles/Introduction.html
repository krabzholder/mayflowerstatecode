The Mayflower State Code contains all laws, regulations, and ordinances that carry the force of law in the State of Mayflower. It is maintained by the Office of the State Registrar, a developer-appointed position responsible for timely codification of logs into this spreadsheet. The Community Administration directives regulating the State Registrar are found at 3 M.S.C. 1 § 3101 et seq. in Title 3. A copyable version of the Mayflower State Code is available on GitHub Pages. All updates to the Mayflower State Code are published in the #mayflower-gazette channel in the State Legislature discord server.

If a law, act, or ordinance is not codified within this sheet, it does not carry the force of law and it is unenforcable.
